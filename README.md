# Unity Summer Mini Project

## 프로젝트 소개
Unity와 C#을 사용해 제작하는 탑다운 전투 프로토타입입니다.

## 개발 목적
Unity 클라이언트 개발자로서 필요한 기본 시스템을 직접 구현하고,
전투, UI, 데이터 관리, 최적화 과정을 포트폴리오로 정리하는 것을 목표로 합니다.

## 사용 기술
- Unity
- C#
- UGUI
- ScriptableObject
- JSON Save/Load 예정

## 구현 예정 기능
- 플레이어 이동
- 카메라 추적
- 적 AI
- 체력/데미지 시스템
- 전투 UI
- 아이템/인벤토리
- 스킬 시스템
- 오브젝트 풀링
- 저장/로드

### Day 0
- 프로젝트 생성
